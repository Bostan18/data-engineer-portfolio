# Projet 04 : Scraping et stockage de donnees web

## Contexte metier

Une plateforme e-commerce analyse les catalogues de livres pour mieux comprendre les tendances de prix et de disponibilite.

## Contexte technique

Scraping du site books.toscrape.com avec BeautifulSoup, nettoyage des doublons, et stockage dans une base SQLite.

## Technologies

Python 3.12, BeautifulSoup4, requests, SQLite3
